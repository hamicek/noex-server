## Kapitoly

### [12.1 Dashboard v reálném čase](./01-dashboard-v-realnem-case.md)

Stavba dashboardu s živými metrikami:
- Reaktivní dotazy pro real-time datové streamy

### [12.2 Chatovací aplikace](./02-chatovaci-aplikace.md)

Stavba multi-client chatovacího systému:
- Store pro persistenci zpráv
- Rules pro zpracování eventů (indikátory psaní, potvrzení přečtení)
- Multi-client push pro real-time doručování
- Transakce pro konzistentní aktualizace stavu

### [12.3 E-commerce backend](./03-ecommerce-backend.md)

Stavba kompletního e-commerce backendu:
- Všechny features spolupracující
- Produkční konfigurace (auth, rate limiting, heartbeat, backpressure)
- Multi-bucket transakce pro zpracování objednávek
